# Dados locais

Esta pasta não distribui dados. Bases cruas e processadas linha a linha são sensíveis, permanecem ignoradas pelo Git e não devem ser publicadas.

Para obter acesso autorizado, posicionar o arquivo e validar a cópia local, consulte:

- `docs/obtencao_dados.md`;
- `docs/reprodutibilidade.md`.

O projeto não baixa a base automaticamente e não fornece URL restrita.
